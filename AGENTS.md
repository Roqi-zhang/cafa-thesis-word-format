# Agent Instructions

This repository contains a reusable Codex Skill, not thesis content.

## Required Workflow

1. Understand the task.
2. Inspect only what is needed.
3. Produce a plan.
4. Wait for explicit human confirmation before modifying Word documents.
5. Execute only the confirmed scope.
6. Validate and report.

## Repository Safety

- Do not add real thesis files.
- Do not add generated reports.
- Do not add school source-document copies.
- Do not add personal identifiers.
- Do not add private filesystem paths.
- Do not run `git commit`, `git push`, or upload to GitHub without a confirmed file list and remote repository.

## Skill Entry Point

`SKILL.md` is the only required entry point. `agents/openai.yaml` is optional metadata only.
